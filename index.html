FMB+

Flexible Motion Brush Plus — Gesture authoring for virtual environments

Abstract

FMB+ is a gesture authoring interface that allows for expressive motion capture and annotation inside virtual environments. Built for real-time systems, FMB+ enables interactive design, modification, and playback of gesture-based interactions using intuitive brush-based controls.
